# Actions
The actions are the main part of the bot. They are used to perform actions when a trigger is met. You can find more information about how to use them [here](/itsmybot/scripting/).

---
# All Actions

## addReaction
Add a reaction to a message. The reaction can be an emoji or a custom emoji.

```yaml
- id: addReaction
  args:
    value: "✅"
```

## addRole
Add a role to a member. The role can be a string or an array of strings.

```yaml
- id: addRole
  args:
    value: "Admin" # Role ID or name
- id: addRole
  args:
    value:
      - "Admin" # Role ID or name
      - "Moderator" # Role ID or name
```

## deleteMessage
Delete a message.

```yaml
- id: deleteMessage
```

## pinMessage
Pin a message.

```yaml
- id: pinMessage
```

## removeReaction
Remove a reaction from a message. The reaction can be an emoji or a custom emoji. If no value is provided, every reaction is removed.

```yaml
- id: removeReaction
  args:
    value: "✅"
```

## removeRole
Remove a role from a member. The role can be a string or an array of strings.

```yaml
- id: removeRole
  args:
    value: "Admin" # Role ID or name
- id: removeRole
  args:
    value:
      - "Admin" # Role ID or name
      - "Moderator" # Role ID or name
```

## reply
Reply to a message or an interaction. You can see the [message configuration](/itsmybot/configuration/message) for more information.
**Can triggers actions**

```yaml
- id: reply
  args:
    content: "Hello there !"
```
## sendMessage
Send a message. You can see the [message configuration](/itsmybot/configuration/message) for more information.
**Can triggers actions**

```yaml
- id: sendMessage
  args:
    embeds:
      - title: "Hello there !"
        description: "This is a test message"
        color: "#00ff00"
```

## editMessage
Edit a message. You can see the [message configuration](/itsmybot/configuration/message) for more information.

```yaml
- id: editMessage
  args:
    content: "Hello there !"
    embeds:
      - title: "This is an edit"
        description: "This is a test message"
```

## sendPrivateMessage
Send a private message. You can see the [message configuration](/itsmybot/configuration/message) for more information.

```yaml
- id: sendPrivateMessage
  args:
    content: "Hello %user_mention% !"
```

## sendRequest
Send a http request. New placeholders will be added when you trigger an action after this action. With the format `%data_<object>%`. If you want a value that is inside an object, you can add the path with underscore `_`. For example, if you want the value of `user.id`, you can use `%data_user_id%`.
***Can triggers actions**
<CodeGroup>
```yaml {{ title: 'Usage' }}
- id: sendRequest
  args:
    value: 'https://api.example.com' # URL
    method: 'POST' # Method, optional. Default is GET
    headers: # Optional
      Content-Type: 'application/json' # Headers
    body: # Optional
      key: 'value' # Body
```

```yaml {{ title: 'Example' }}
- id: sendRequest
  args:
    value: 'https://api.mcstatus.io/v2/status/java/mc.hypixel.net:25565' # URL
    actions:
      - id: sendMessage
        args:
          content: "Players %data_players_online%/%data_players_max% | Version %data_version_name_clean%"
```
</CodeGroup>

## startThread
Start a thread. If the value is a array, a random value will be selected.
***Can triggers actions**

```yaml
- id: startThread
  args:
    value: "Hello there !"
- id: startThread
  args:
    value:
      - "Thread 1"
      - "Thread 2"
```

## unpinMessage
Unpin a message.

```yaml
- id: unpinMessage
```

## openForm
Open a form. Requires the Forms addon.

```yaml
- id: openForm
  args:
    value: "form_id" # Form ID
```

## addInviteBonus
Add bonus invites to a user. Requires the [Invite Traker](/itsmybot/addons/invites/) addon.

```yaml
- id: addInviteBonus
  args:
    amount: 5 # Amount of bonus invites
```

## removeInviteBonus
Remove bonus invites to a user. Requires the [Invite Traker](/itsmybot/addons/invites/) addon.

```yaml
- id: removeInviteBonus
  args:
    amount: 5 # Amount of bonus invites
```
## setInviteBonus
Set bonus invites to a user. Requires the [Invite Traker](/itsmybot/addons/invites/) addon.

```yaml
- id: setInviteBonus
  args:
    amount: 5 # Amount of bonus invites
```
