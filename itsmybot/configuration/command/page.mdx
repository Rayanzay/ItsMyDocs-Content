# Command Properties
All the properties you can use in the message configuration. Every property is optional.

<Properties>
  <Property name="enabled" type="boolean">
    If false, the command will be disabled.
  </Property>
  <Property name="aliases" type="string[]">
    The aliases of the command.
  </Property>
  <Property name="cooldown" type="number">
    The cooldown of the command in seconds.
  </Property>
  <Property name="roles" type="string[]">
    The roles required to use the command.
  </Property>
  <Property name="inherited" type="boolean">
    If true, member with a roles higher than the required roles can use the command.
  </Property>
  <Property name="permissions" type="string[]">
    The permissions required to use the command.
  </Property>
  <Property name="permission" type="string">
    Directly use discord permissions
  </Property>
  <Property name="channels" type="string[]">
    The channels where the command can be used.
  </Property>
  <Property name="users" type="string[]">
    The users who can use the command.
  </Property>
</Properties>

<CodeGroup title="Example">
```yaml {{ title: 'Simple Command' }}
plugin:
  # Configuration can be added here.
  permission: "ADMINISTRATOR" # Here for example, the command will require the administrator permission.
  description: "Manage plugins."
  subcommands:
    list:
      description: "Show the list of plugins."
```

```yaml {{ title: 'Complex Command' }}
server-info:
  # Configuration can be added here.
  aliases: # The command can be used with the aliases "server" and "guild".
    - "server"
    - "guild"
  cooldown: 5 # The command can be used every 5 seconds.
  channels: # The command can only be used in the following channels.
    - "112233445566778899"
    - "command-channel"
  description: "Show information about the server."
```

```yaml {{ title: 'Advanced Command' }}
ban:
  # Configuration can be added here.
  aliases: # The command can be used with the aliase "b".
    - "b"
  roles: # The command requires the following roles.
    - "Moderator"
  inherited: true # Members with a role higher than "Moderator" can use the command.
  permissions: # The command requires the following permissions.
    - "BAN_MEMBERS"
  description: "Ban a member."
```
</CodeGroup>
