# Embeds Plugins
Embeds is a simple plugin that allows you to create embeds and message with a yaml configuration file.

## Create an Embed

Create a new file in the `configs/Embeds/embeds` folder or one of its subfolders. This file will contain the configuration for your message, including the content, embeds, and components.
For the syntax of the configuration file, see the [Message configuration](/itsmybot/configuration/message) page.
